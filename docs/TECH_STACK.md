# Tech Stack Summary

## Backend
- **Python 3.10+**
- **Flask** (web framework)
- **SQLAlchemy** (ORM)
- **PostgreSQL** (database)
- **Google Gemini API** (AI integration)

## Frontend
- **HTML5** (semantic markup)
- **CSS3** + **Tailwind CSS** (via CDN)
- **Vanilla JavaScript** (ES6+, NO frameworks)

## Key Principles
- **NO React, Vue, Angular, or any JS frameworks**
- **NO Node.js** (backend is pure Python/Flask)
- **NO build process** (Tailwind via CDN)
- **Progressive Enhancement** (works without JS)
- **Server-Side Rendering** (Flask/Jinja2)
