# AsesoriaIMSS.io

## Overview
AsesoriaIMSS.io is a platform designed to provide advice and information related to IMSS (Instituto Mexicano del Seguro Social).

## Tech Stack

### Backend
- **Language**: Python 3.10+
- **Framework**: Flask with Blueprints
- **ORM**: SQLAlchemy
- **Database**: PostgreSQL

### Frontend
- **HTML5** (semantic markup)
- **CSS3** with **Tailwind CSS** (via CDN)
- **JavaScript Vanilla** (NO React, NO Vue, NO frameworks)
- **Responsive Design** (mobile-first)
- **Multi-language Support** (ES/EN)

### Infrastructure
- **API Integration**: Google Gemini (centralized)
- **Deployment**: Hostinger VPS ($9.99/mes)

## Setup
1. Clone the repository.
2. Install Python dependencies: `pip install -r requirements.txt`
3. Configure environment variables in `.env`.
4. Run the Flask application: `python run.py`
5. Open browser at `http://localhost:5000`
